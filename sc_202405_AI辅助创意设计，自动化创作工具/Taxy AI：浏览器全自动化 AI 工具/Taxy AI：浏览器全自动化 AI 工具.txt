


            
Taxy AI：浏览器全自动化 AI 工具
          




介绍Taxy是一款非常有用的工具，可以帮助用户自动执行重复操作，如单击按钮或输入文本框。该工具使用了GPT-4技术，可以在用户定义的临时指令下，控制用户的浏览器。不仅如此，它还提供了计划工作流和保存功能的支持，以帮助用户更好地管理其操作过程。虽然Taxy目前还处于研究和开发阶段，但这不影响其使用。如果您想要体验这个工具，可以按照说明在本地运行它。如果您想知道更多关于Taxy的信息，可以注册候补名单，以便随时掌握其最新开发情况。需要注意的是，Taxy是完全开源的，所以用户不必担心隐私问题。这是Taxy使用Google日历，提示“安排明天上午10点站立。邀请 david@taxy.ai”安装和运行目前此扩展程序只能通过 GitHub 存储库获得。我们会在增加针对非技术人员的可用性功能后，在 Chrome 网上应用店发布它。要在您的计算机上构建和安装此扩展程序，请按照以下说明操作。安装拓展请确保您已经安装了 Node.js >= 16。克隆此存储库。运行 yarn 命令安装依赖项。运行 yarn start 命令构建扩展程序包。按照以下步骤在 Chrome 上加载扩展程序：导航到 chrome://extensions/。切换至开发者模式。点击“加载已解压的扩展程序”。选择 yarn start 生成的构建文件夹。运行在你的浏览器1. 安装完成后，浏览器插件将以两种形式提供：    1.1 弹出窗口形式。您可以通过在 Mac 上按下 cmd+shift+y，在 Windows/Linux 上按下 ctrl+shift+y 或单击浏览器中的扩展程序图标来激活它。       1.2 开发工具面板形式。您需要先打开浏览器的开发工具，然后导航到 Taxy AI 面板来激活它。2. 接下来您需要做的是创建或访问现有的 OpenAI API 密钥，并将其粘贴到提供的框中。此密钥将安全地存储在您的浏览器中，不会上传到第三方。3. 最后，导航到您想要在 Taxy 上测试的网页（例如 OpenAI 编程环境）并开始尝试！Taxy 运行原理Taxy可以在网页上提取重要的元素，然后将其模板化。Taxy会将简化后的网页和用户指令发送给LLM（语言模型），LLM可以根据用户指令执行交互操作。当Taxy从LLM获取结果时，它会解析响应以获取操作，并使用Chrome调试器API执行这些操作。如果出现任何错误或者用户停止了任务的执行，操作循环将在此阶段结束。目前，Taxy可以完成最多50个操作的单个任务。
