创作内容与人工智能生成内容

王宏飞｜湖北经济学院数字媒体艺术系

人们在网络上经常提及AIGC。那么，AIGC到底是什么呢？AIGC全称是Artificial Intelligence  Generated Content，亦即人工智能创作内容。

什么是创作内容？创作内容也就是生产内容，比如写文章、写诗、唱歌、创造知识都是创造内容的表现形式。与AIGC相对应的概念还有PGC、UGC、PUGC。按照发展顺序，创作内容合计走过了四个阶段。

首先是PGC，全称是Professional Generated Content，翻译过来就是专家生产内容。专家生产内容模式，主要表现为由专家或专业机构来进行内容的生产，他们具备专业的内容生产能力，能够保证内容的专业性，特指生产非常专业的内容或知识。主要应用在知识付费、在线教育、学习平台等专业平台或企业。

其次是UGC，全称是User Generated Content，翻译过来就是用户生成内容，即用户原创内容。用户生成内容模式，主要以用户的需求为起点，用户创作内容后可以通过互联网平台进行展示、发布，可以让更多的人看到，帮助别人的同时自己也获得了成长，甚至进行知识变现。比如每天都有人在抖音号、视频号上发布原创视频，在公众号、知乎上发布原创文章，或者在平台上回答别人的提问。

再次是PUGC，全称是Professional User Generated Content，即专业用户生成内容，也就是专家用户生产内容模式，主要以专业用户为内容起点，是内容生产者和用户的结合，主要内容形式是将PGC与UGC相结合，但相比较PGC而言，显得更加专业，能够满足更多的个性化需求，比如互联网上的情感咨询、医疗问答等内容，回答问题的专家是经过认证的专业认识；再比如一群专业的短视频创作者，他们在互联网平台发布现场创作的有针对性的30秒店面推广短视频，进而扩大自己的影响力，最后通过视频种草来实现才艺变现。

最后就是AIGC，也就是人工智能技术生成内容。一些公司开发的人工智能模型可以较为专业地回答用户提出的问题，或者根据用户的需求生成相关内容，而且具有较高的参考价值。所以AIGC可以用来辅助进行内容生产，成为一种新型创作工具，可以提升工作效率。比如ChatGPT, Midjourney,能实现文生文、文生图、图生图。目前AIGC可应用于文字创作、音频创作、绘画创作、视频创作、代码生成、游戏开发等领域。

1999年，我参加高考。当年的语文作文题是“假如记忆可以移植……”。如今借助信息技术，人类积累的知识能以数据的方式存储在硬盘里，人类的知识数据实现了跨越时间和空间进行积累并高效输出，以一种数据化的方式基本实现了记忆的移植。与此同时，人工智能模型可以学习人类积累的所有知识，并模仿人进行思考、创作，进而为人类提出的问题生成有理有据的解决方案。人工智能生成内容这种内容创作层面的技术革命将极大的提升人类的生产效率，这种变革能媲美将煤炭资源转化为能稳定输出的动力式的革命。人类之前的蒸汽革命将人类从繁重体力劳动中解放出来，如今的人工智能革命即将把人类从繁重的脑力劳动中解放出来。
